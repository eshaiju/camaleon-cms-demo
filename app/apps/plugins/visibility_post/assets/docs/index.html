Description
Posts Visibility allows users to set the visibility of an article, which can be public, private or password protection, setting an article as public then any user can access the item, if the item is set to private so we have options select some user group, the user group is selected you will see the publication of the article and not the other groups as well.

Example: If the Private option is selected and 2 user groups Administrator, Customer have and the user group Administrator visibility item is selected, then the user group client can not see the publication of the article

Configuration

After activating the plugin, select the "Content / Post / Add New" to set the visibility of the Post, go to the Settings panel, the right sidebar

Select a date for the publication of post

Types:
1.- Public: Everyone can see post
2.- Private: Only a few can see the post
3.- Password protection: put a password to see the post
