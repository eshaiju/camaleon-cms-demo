Description
Contact Form plugin allows you to create multiple contact forms which can be used by the shortcode that each contact form, this shortcode is copied and pasted into the content of a post, then when visiting this post is achieved vizualizar form I created.

Contact Form Plugin has the following characteristics:
  • Detail Form Panel: Form essentials
  • Panel Edit Fields: edit/delete form fields dynamically
  • Panel Add Fields: which has 2 tabs (Add Fields and Edit Fields) form fields dynamically
  • Email Settings Panel Form: setting fields for sending mail data
  • Messages Settings Panel form: set the basic messages of the form fields

 
Configuration
Select the link "Settings" to open the menu and find the link to the plugin " Contact Form "
Click on the link " Contact Form " to access the list of forms and also to create forms.
Selecting the link " Contact Form ", proceed to create a form by placing a Name Description , then click on the button "Submit" .
Selecting the button "Submit" , the following screen where we have the following configuration fields appear:

1. Form Details: In this panel we form fields to edit Name , Slug .
2. Settings Form Email: in this panel settings have fields for email sending system, where
  • "To:" allows you to enter the email which came messages of the form
  • "Subject:" here the email subject is placed,
  • "Body:" Here the message body is placed.
3. Settings Form Messages: in this panel settings have error messages / confirmation of the form, each field can be enter the message you want.
4. Edit Field: In this panel all fields on the form you want it placed, plus you have the option to rename the form submit button in the "Name Button" field.
5. Add Field: In this panel you have all the fields for a form, how to add fields to Edit Field panel is as follows:
In the list of elements of the plug contact form shortcode we copy the contents column that has the selected item such as [forms slug = contact-form] , then copy this value, any Post and habre paste the heat content of a post and saved. Habre this post later in the frontend to display the form created.
